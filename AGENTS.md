
## Preact

* Use class Preact components. Only use function Preact components for stateless components.

## TypeScript

* Timestamp fields should end in "Time". Examples: createTime; and purchaseTime.
* Boolean fields should start with words like "is", "should", "has", "can", etc. Examples: isPrivate; shouldShowFooButton.
* Every TypeScript file should have an @file JSDoc comment. The comment should be useful to someone new to the codebase that might be wondering, "what's the purpose of this file and its contents?".

## Other

* Git commit messages should be in imperative present tense.
* Do not use conventional commit messages.
